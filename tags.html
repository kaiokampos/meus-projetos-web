Guia de Tags HTML

Este documento lista as principais tags HTML, suas funções e exemplos práticos para consulta durante o aprendizado de desenvolvimento web.

Estrutura Básica

  • <!DOCTYPE html>:

    Declara que o documento é HTML5. Vai no topo.

    Exemplo: <!DOCTYPE html> (sem fechamento).

  • <html>:

    Contém todo o conteúdo da página. Define idioma com lang.

    Exemplo: <html lang="pt-BR">...</html>.

  • <head>:

    Inclui metadados (título, CSS, etc.).

    Exemplo: <head><title>Minha Página</title></head>.

  • <body>:

    Contém o conteúdo visível (textos, imagens, etc.).

    Exemplo: <body><h1>Olá</h1></body>.

Metadados (dentro de <head>)

  • <meta>:

    Define metadados, como codificação ou responsividade.

    Exemplo: <meta charset="UTF-8">, <meta name="viewport" content="width=device-width, initial-scale=1.0">.

  • <title>:

    Título da aba do navegador.

    Exemplo: <title>Meu Site</title>.

  • <link>:

    Conecta arquivos CSS.

    Exemplo: <link rel="stylesheet" href="styles.css">.

  • <script>:

    Inclui JavaScript.

    Exemplo: <script src="script.js"></script>.

Conteúdo de Texto

  • <h1> a <h6>:

    Títulos, do maior ao menor.

    Exemplo: <h1>Título Principal</h1>.

  • <p>:

    Parágrafo.

    Exemplo: <p>Este é um texto.</p>.

  • <strong>:

    Texto em negrito (semântico).

    Exemplo: <strong>Atenção!</strong>.

  • <em>:

    Texto em itálico (semântico).

    Exemplo: <em>Ênfase</em>.

  • <span>:

    Agrupa texto para estilização.

    Exemplo: <span class="destaque">Texto</span>.

Listas

  • <ul>:

    Lista não ordenada (marcadores).

    Exemplo: <ul><li>Item 1</li><li>Item 2</li></ul>.

  • <ol>:

    Lista ordenada (números).

    Exemplo: <ol><li>Passo 1</li><li>Passo 2</li></ol>.

  • <li>:

    Item de lista.

    Exemplo: <li>Maçã</li>.

Links e Mídia

  • <a>:

    Cria links.

    Exemplo: <a href="https://google.com">Google</a>, <a href="#contato">Contato</a>.

  • <img>:

    Insere imagens.

    Exemplo: <img src="foto.jpg" alt="Descrição">.

  • <video>:

    Incorpora vídeos.

    Exemplo: <video src="video.mp4" controls></video>.

  • <audio>:

    Incorpora áudios.

    Exemplo: <audio src="audio.mp3" controls></audio>.

Estrutura Semântica

  • <header>:

    Cabeçalho da página.

    Exemplo: <header><h1>Meu Site</h1></header>.

  • <nav>:

    Menu de navegação.

    Exemplo: <nav><a href="#home">Home</a></nav>.

  • <main>:

    Conteúdo principal.

    Exemplo: <main><p>Conteúdo</p></main>.

  • <section>:

    Seção temática.

    Exemplo: <section><h2>Sobre</h2></section>.

  • <article>:

    Conteúdo independente.

    Exemplo: <article><h2>Post</h2></article>.

  • <aside>:

    Conteúdo lateral.

    Exemplo: <aside><p>Publicidade</p></aside>.

  • <footer>:

    Rodapé.

    Exemplo: <footer><p>© 2025</p></footer>.

Formulários

  • <form>:

    Cria formulários.

    Exemplo: <form action="/enviar"><input type="text"></form>.

  • <input>:

    Campo de entrada.

    Exemplo: <input type="text" name="nome" placeholder="Nome">.

  • <textarea>:

    Área de texto.

    Exemplo: <textarea name="mensagem"></textarea>.

  • <button>:

    Botão.

    Exemplo: <button type="submit">Enviar</button>.

  • <label>:

    Rótulo para campos.

    Exemplo: <label for="nome">Nome:</label><input id="nome">.

Tabelas

  • <table>:

    Cria tabela.

    Exemplo: <table><tr><td>Dado</td></tr></table>.

  • <tr>:

    Linha da tabela.

    Exemplo: <tr><td>1</td></tr>.

  • <td>:

    Célula de dados.

    Exemplo: <td>Valor</td>.

  • <th>:

    Célula de cabeçalho.

    Exemplo: <th>Nome</th>.

Outras

  • <div>:

    Contêiner genérico.

    Exemplo: <div class="container"><p>Conteúdo</p></div>.

  • <br>:

    Quebra de linha.

    Exemplo: <p>Linha 1<br>Linha 2</p>.

  • <hr>:

    Linha horizontal.

    Exemplo: <hr>.

Conteúdo Semântico Avançado

  • <figure>:

    Agrupa mídia com legenda.

    Exemplo: <figure><img src="foto.jpg" alt="Paisagem"><figcaption>Foto</figcaption></figure>.

  • <figcaption>:

    Legenda para figure.

    Exemplo: <figcaption>Descrição</figcaption>.

  • <details>:

    Conteúdo expansível.

    Exemplo: <details><summary>Clique</summary><p>Conteúdo</p></details>.

  • <summary>:

    Título para details.

    Exemplo: <summary>Ver mais</summary>.

Formulários Avançados

  • <select>:

    Menu dropdown.

    Exemplo: <select name="opcoes"><option value="1">Opção 1</option></select>.

  • <option>:

    Item de select.

    Exemplo: <option value="1">Opção 1</option>.

  • <datalist>:

    Sugestões para input.

    Exemplo: <input list="sugestoes"><datalist id="sugestoes"><option value="Opção 1"></datalist>.

  • <output>:

    Resultados de cálculos.

    Exemplo: <output name="resultado">0</output>.

Mídia e Gráficos

  • <canvas>:

    Gráficos com JavaScript.

    Exemplo: <canvas id="meuCanvas" width="200" height="100"></canvas>.

  • <svg>:

    Gráficos vetoriais.

    Exemplo: <svg><circle cx="50" cy="50" r="40" fill="blue"></circle></svg>.

  • <iframe>:

    Conteúdo externo.

    Exemplo: <iframe src="https://www.youtube.com/embed/VIDEO_ID" width="560" height="315"></iframe>.

Outras

  • <progress>:

    Barra de progresso.

    Exemplo: <progress value="70" max="100">70%</progress>.

  • <meter>:

    Valor em intervalo.

    Exemplo: <meter value="0.6" min="0" max="1">60%</meter>.

  • <time>:

    Datas ou horários.

    Exemplo: <time datetime="2025-05-16">16 de Maio de 2025</time>.

  • <mark>:

    Destaca texto.

    Exemplo: <mark>Texto destacado</mark>.
