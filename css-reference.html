Guia de Propriedades CSS

Este documento lista as principais propriedades CSS, suas funções e exemplos práticos para consulta durante o aprendizado de desenvolvimento web.

Seletores

  • Seletor de Tag:

    Aplica estilos a uma tag HTML específica.

    Exemplo: h1 { color: blue; }

    Demonstração:

    Título Azul

  • Seletor de Classe:

    Aplica estilos a elementos com uma classe específica.

    Exemplo: .destaque { background-color: yellow; }

    Demonstração: Texto Destacado

  • Seletor de ID:

    Aplica estilos a um elemento com um ID específico.

    Exemplo: #unico { font-size: 20px; }

    Demonstração:

    Texto Único

Cores

  • color:

    Define a cor do texto.

    Exemplo: p { color: #FF0000; }

    Demonstração:

    Texto Vermelho

  • background-color:

    Define a cor de fundo.

    Exemplo: div { background-color: rgb(0, 255, 0); }

    Demonstração:

    Fundo Verde

Propriedades de Texto

  • font-family:

    Define a fonte do texto.

    Exemplo: p { font-family: Arial, sans-serif; }

    Demonstração:

    Texto em Arial

  • font-size:

    Define o tamanho da fonte.

    Exemplo: h2 { font-size: 24px; }

    Demonstração:

    Título Grande

  • font-weight:

    Define a espessura da fonte.

    Exemplo: span { font-weight: bold; }

    Demonstração: Texto Negrito

  • text-align:

    Alinha o texto.

    Exemplo: p { text-align: center; }

    Demonstração:

    Texto Centralizado

  • text-decoration:

    Adiciona decoração ao texto (sublinhado, riscado).

    Exemplo: a { text-decoration: none; }

    Demonstração: Link sem Sublinhado

Modelo de Caixa

  • margin:

    Define a margem externa.

    Exemplo: div { margin: 20px; }

    Demonstração:

    Caixa com Margem

  • padding:

    Define o preenchimento interno.

    Exemplo: div { padding: 15px; }

    Demonstração:

    Caixa com Preenchimento

  • border:

    Define bordas.

    Exemplo: div { border: 2px solid blue; }

    Demonstração:

    Caixa com Borda

  • width:

    Define a largura.

    Exemplo: div { width: 200px; }

    Demonstração:

    Largura 200px

  • height:

    Define a altura.

    Exemplo: div { height: 100px; }

    Demonstração:

    Altura 100px

  • box-sizing:

    Controla como largura e altura são calculadas.

    Exemplo: div { box-sizing: border-box; }

    Demonstração:

    Caixa com Box-Sizing

Exibição e Posicionamento

  • display:

    Define como o elemento é exibido.

    Exemplo: span { display: inline-block; }

    Demonstração: Inline-Block

  • position:

    Define o método de posicionamento.

    Exemplo: div { position: relative; top: 10px; }

    Demonstração:

    Deslocado 10px
